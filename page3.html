เปิ้นก็พูดไม่ค่อยเก่งหรอกนะงับ : ) ก็อยากจะบอกซันว่า....

เปิ้นขอขอบคุณสำหรับmine set ดีๆที่คอยแชร์ให้กันนะงับ
เปิ้นชอบฟังซันเล่าเรื่องหรืออธิบายบางสิ่งบางอย่างมาก
เป็นเด็กจิตวิทยาป่ะ55 ( มีหนังสือแบ่งกันอ่านบ้างนะงับ อยากอ่านเดี๋ยวตามคนไม่ทันละ ) คือฟังได้เพลินๆ เข้าใจบ้างไม่เข้าใจบ้าง แบบบางเรื่องก็รู้สึกว้าวมาก ถ้าไม่เข้าใจละเวลาเปิ้นถามซัน
ซันก็มักจะตอบและอธิบายละเอียดอีกครั้ง ซึ่งเปิ้นก็ไม่ค่อยได้จำหรอกงับ เลยฟังซันเล่าได้หลายรอบ55

ขอบคุณที่เป็นรอยยิ้มให้เปิ้นนะงับ แม้บางครั้งเปิ้นก็เมินใส่บ้าง
(ดูเห็นแก่ตัวใช่ไหม) ต้องขอโทษด้วย
ก็ขอให้ซันเป็นรอยยิ้มให้คนรอบตัวแบบนี้ต่อไปนะงับ ยิ้มเยอะๆน้า
ละก็ขอโทษนะงับที่กวนซัน ถ้าซันไม่อ่านไม่ตอบ เปิ้นก็ไม่ได้ว่าไรเลยงับ :)
และต่อจากนี้มีเรื่องอะไร ก็ทักมาได้เหมือนเดิมนะงับ เปิ้นไม่ได้หายไปไหน หรือโทรมาก็ได้เช่นกันงับ
